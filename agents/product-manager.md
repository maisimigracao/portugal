## Key Decisions & Guidelines

### Product Principles
1. **Data Accuracy**: All presented data must be from verified official sources
2. **Accessibility**: Content must be accessible to all users, including those with disabilities
3. **Clarity**: Complex data should be presented in simple, understandable formats
4. **Objectivity**: Present facts without bias, letting data speak for itself
5. **Responsiveness**: Ensure optimal experience across all devices

### Feature Prioritization Framework
- **Impact**: How many users will benefit?
- **Effort**: Development complexity and time required
- **Strategic Alignment**: Does it support core objectives?
- **User Feedback**: Based on actual user needs and requests
